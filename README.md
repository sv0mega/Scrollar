# Scrollar

Scrollar is a research discovery and summarization platform that helps users explore scientific papers through concise, engaging summaries and insights.

## Features

* Search and discover research papers
* AI-powered paper summaries
* Trending research section
* Save papers to a personal library
* Responsive modern user interface
* Fast performance powered by React and Vite

## Tech Stack

* React
* TypeScript
* Vite
* Node.js

## Getting Started

### Prerequisites

* Node.js (v18 or later)
* npm

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

### Build for Production

```bash
npm run build
```
